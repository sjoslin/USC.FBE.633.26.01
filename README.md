# FBE 633 - Coding Agents Module

This is the course repository for the coding agents supplementary module for FBE 633 (Asset Pricing Theory).

## Course Overview

This supplementary module covers the usage of coding agents and related tools for empirical analysis in asset pricing. The purpose is to equip students with practical implementation skills to complement the core theoretical material.

## What We'll Cover

- **Coding agents**: How to use agents that write, execute, and debug code autonomously
- **Guardrails for autonomous agents**: Architectural controls for safe agent deployment
- **Data organization**: Accessing FRED, WRDS, and other data sources programmatically
- **Key tools**: Docker, Git, VS Code with agent access
- **LaTeX**: Publication-ready research documents with agent aid

## Structure

| Week | Topic |
|--|----|
