# Topic 4 - Sandboxing

## Overview

We will cover sandboxing in this topic. We will show how to use docker to create containers to isolate files and connection to the outside. We will discuss threat models and discuss the safeguards that a sandbox + git together provide. This combination works to greatly enhance the safety of 'dangerously skip mode' which allows for automation of complex workflows.
