# Topic 3 - Git + Data Organization

## Overview

In this topic, we will discuss Git and Data Organization. Git acts as your undo button to allow you to effectively collaborate with AI agents to generate code safely. We also talk about properly setting up project directories such as keeping secrets, warehousing raw data, and creating persistent context instructions with CLAUDE.md files.
